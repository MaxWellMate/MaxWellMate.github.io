hexo博客 Matery主题 Typora 写文章图片显示问题(一)

问题：在使用typora 写hexo博客时，部署到Github端无法显示。

经过一番折腾，发现本地和github只能活一个

先写一个部署到github 活，后面写本地活。

解决方案：

前提：安装img插件 

post_asset_folder: true   # 打开 _config.yml  修改对应的数据 fasle->true
npm install hexo-asset-img --save
第一步：在_posts 文件夹下创建 test.md 和 test文件夹（存放图片）
第二部：设置typora  文件——文件——偏好设置——图像，优先使用相对路径

第三步：插入图片
第四步: 
hexo clean
hexo g  #(此时生成的对应的index.html文件是不显示图片的)
hexo d  #网页端查看内容
